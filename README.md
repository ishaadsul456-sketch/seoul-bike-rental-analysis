# Seoul Bike Rental Analysis

## About
Analyzed Seoul bike sharing data to understand how weather 
and time affect bike rental demand.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- NumPy

## Key Insights
- Rentals highest in Summer and Autumn
- Peak rental hours are 8 AM and 8 PM
- Temperature positively affects rentals
- Humidity and Rainfall reduce rentals

## Files
- Seoul_Rented_Bike.ipynb — main analysis notebook
- SeoulBikeData.csv — dataset
